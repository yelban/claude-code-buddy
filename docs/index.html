Quick Start

Get up and running in 2 minutes

1

Prerequisites

Node.js 18+ and Claude Code installed

# Check Node.js version
node --version  # Should be v18 or higher
2

Clone & Install

# Clone the repository
git clone https://github.com/PCIRCLE-AI/claude-code-buddy.git
cd claude-code-buddy

# Run interactive installer (handles everything)
./scripts/install.sh
3

Start Using

Restart Claude Code and try these buddy commands:

"Analyze my codebase architecture"
"Generate tests for auth.ts"
"Review this code for security"
"Optimize this database query"

No API keys needed for core features - CCB uses your existing Claude Code subscription.

Optional: RAG (Retrieval-Augmented Generation) feature requires an OpenAI API key for enhanced codebase search capabilities.
