隱私權政策

最後更新：2026 年 1 月 3 日

概述

Claude Code Buddy (CCB) 是一個完全在本機運行的開源 MCP 伺服器。您的隱私至關重要 - 我們設計 CCB 讓您的資料保持在自己的掌控之下。

資料儲存

僅本機儲存： 所有 CCB 資料都儲存在您的電腦本機：

  • data/knowledge-graph.db - 您的專案知識和記憶
  • logs/ - 除錯和效能日誌

您擁有您的資料： CCB 從不將您的程式碼、專案資料或知識圖譜傳送到外部伺服器。一切都保留在您的電腦上。

Claude API： 當 CCB 將任務路由到特定能力時，它會透過您現有的 Claude Code 訂閱與 Claude 通訊。這受 Anthropic 隱私權政策 規範。

網站 (ccb.pcircle.ai)

無分析工具： 我們的文件網站不使用追蹤、分析或 Cookie。

Google Fonts： 網站使用 Google Fonts 顯示文字。Google 可能會收集您的 IP 位址作為字型傳送的一部分。請參閱 Google 隱私權政策。

您的控制權

  • 隨時刪除 data/ 目錄即可移除 CCB 資料
  • 所有資料處理都在您的電腦本機進行
  • 開源程式碼代表完全透明 - 查看原始碼
