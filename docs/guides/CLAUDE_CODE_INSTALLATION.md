# Claude Code Installation Guide

**Just tell Claude Code**: "Install claude-code-buddy MCP from https://github.com/PCIRCLE-AI/claude-code-buddy"

That's it! Claude Code handles everything automatically.

---

## What Happens During Installation

Claude Code will:

1. **Clone and Setup** - Download and build the project
2. **Configure MCP Server** - Add to Claude Code configuration
3. **Verify** - Run tests and confirm everything works

**Setup time:** 2-5 minutes

---

## After Installation

**Restart Claude Code**, then you can:

```
"List available Claude Code Buddy tools"
"Make code review focus on security"
"Create a capability for API documentation"
"Show me the system architecture"
```

---

## Troubleshooting

**Having issues?** Just ask Claude Code:

```
"Claude Code can't find claude-code-buddy MCP server"
"Show me claude-code-buddy logs"
"Verify claude-code-buddy installation"
```

Claude Code will diagnose and fix the issue.

---

After installation, customize **anything** by talking to Claude Code:

| What You Want | Just Ask |
