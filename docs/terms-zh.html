服務條款

最後更新：2026 年 1 月 3 日

1. 開源授權

Claude Code Buddy 是根據 AGPL-3.0 授權的免費開源軟體。

您可以自由使用、修改和分發 CCB，但需遵守 AGPL-3.0 授權條款。

2. 無保證聲明

CCB 以「現狀」提供，不提供任何明示或默示的保證。我們不保證：

  • CCB 將無錯誤或不中斷運行
  • AI 生成的程式碼或建議將是準確或合適的
  • CCB 將適用於所有配置或系統
  • 與未來 Claude Code 或 API 版本的相容性

3. 責任限制

CCB 開發者和貢獻者不對您使用 CCB 造成的任何損害負責，包括但不限於：

  • AI 建議導致的程式碼錯誤或 bug
  • 資料遺失或損壞
  • 系統停機或效能問題
  • 安全漏洞
  • Claude 的 API 成本

4. AI 生成內容

CCB 使用 AI (Claude) 生成程式碼、文件和建議。您確認：

  • AI 輸出可能包含錯誤、bug 或安全問題
  • 您有責任審查和測試所有 AI 生成的程式碼
  • AI 建議不應在生產環境中盲目信任
  • 您擁有使用 CCB 生成的所有輸出

5. 您的責任

使用 CCB 時，您同意：

  • 在部署前審查和測試所有 AI 生成的程式碼
  • 使用 Claude API 時遵守 Anthropic 的可接受使用政策
  • 尊重您分享或貢獻的程式碼中的智慧財產權
  • 不將 CCB 用於非法或有害目的
  • 維護重要資料的備份

6. 第三方服務

CCB 整合第三方服務：

  • Claude API： 受 Anthropic 條款 約束

7. 貢獻

透過向 CCB 貢獻程式碼、文件或 issue，您同意您的貢獻將根據 AGPL-3.0 授權。請參閱我們的貢獻指南。

8. 條款變更

我們可能會更新這些條款。變更將在此處發布並更新日期。變更後繼續使用即表示接受。
