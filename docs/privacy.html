Privacy Policy

Last updated: January 3, 2026

Overview

Claude Code Buddy (CCB) is an open-source MCP server that runs entirely on your local machine. Your privacy is paramount - we designed CCB to keep your data under your control.

Data Storage

Local Storage Only: All CCB data is stored locally on your machine in:

  • data/knowledge-graph.db - Your project knowledge and memory
  • logs/ - Debug and performance logs

You own your data: CCB never sends your code, project data, or knowledge graph to external servers. Everything stays on your computer.

Claude API: When CCB routes tasks to specific capabilities, it communicates with Claude through your existing Claude Code subscription. This is governed by Anthropic's Privacy Policy.

Website (ccb.pcircle.ai)

No Analytics: Our documentation website does not use tracking, analytics, or cookies.

Google Fonts: The website uses Google Fonts for typography. Google may collect your IP address as part of font delivery. See Google's Privacy Policy.

Your Control

  • Delete CCB data anytime by removing the data/ directory
  • All data processing happens locally on your machine
  • Open-source code means full transparency - review the source

Contact

Questions? Open an issue on GitHub.

Updates
