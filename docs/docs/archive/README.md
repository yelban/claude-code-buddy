# Archive

This directory contains archived documentation that is no longer actively maintained but kept for historical reference.

**Note**: Contents of this directory are not version controlled (see `.gitignore`).

## What belongs here:
- Outdated design documents
- Superseded planning docs
- Historical evolution records
- Deprecated implementation notes

## What doesn't belong here:
- Current documentation (belongs in `/docs/`)
- Active plans (belongs in `/docs/plans/`)
- Implementation guides (belongs in `/docs/guides/`)
