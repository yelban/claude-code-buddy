### 6.3 FAQ (10 Common Questions)

**Q1: What's the difference between smart-agents and regular Claude Code?**
A: Regular Claude Code has one AI assistant that works sequentially. Smart-agents gives you a team of specialized AI agents that can work in parallel in the background while you continue coding. Think of it as upgrading from a solo assistant to a full engineering team.

**Q2: Will background agents slow down my computer?**
A: No. Smart-agents automatically limits resource usage (70% CPU, 8GB RAM by default) and monitors system health. If resources are low, it automatically pauses background tasks or suggests foreground execution.

**Q3: How do I know which agent did what?**
A: Every action has clear attribution. Success messages show which agent completed what task and how long it took. Error messages clearly state which agent caused an issue and what went wrong.

**Q4: Can I trust the agents to work without supervision?**
A: Smart-agents is designed for transparency and control. You can monitor progress in real-time, cancel tasks anytime, and review all results. The system learns from your corrections to improve accuracy.

**Q5: What happens if an agent makes a mistake?**
A: Agents clearly report errors with explanations. For serious issues, smart-agents automatically rolls back changes and disables the problematic agent. You're always in control.

**Q6: How much faster will I be with smart-agents?**
A: Users typically see 30-50% productivity gains from running tasks in parallel. The actual benefit depends on your workflowâ€”tasks like research, code review, and testing benefit most from background execution.

**Q7: Does smart-agents work offline?**
A: Background execution and progress tracking work offline. However, agents that require API calls (research, certain code analysis) need internet connectivity.

**Q8: Can I use smart-agents with my team?**
A: Currently smart-agents is designed for individual use. Team collaboration features (shared agent pools, collaborative queues) are coming in v2.0.

**Q9: How does the learning system work?**
A: Smart-agents tracks your choices (background vs foreground, which agents you prefer, corrections you make) and adapts suggestions over time. All learning is localâ€”your data never leaves your machine.

**Q10: What's the cost?**
A: Smart-agents is included in Claude Code Pro subscription ($20/month). There are no additional API costsâ€”all agents use your existing Claude API quota.
