## Brand Voice Checklist

Before publishing any message, ask:

- [ ] Is it **authentic** (not corporate-speak)?
- [ ] Is it **empowering** (user in control)?
- [ ] Is it **transparent** (honest about limitations)?
- [ ] Is it **collaborative** (working together)?
- [ ] Is it **accessible** (no unnecessary jargon)?
